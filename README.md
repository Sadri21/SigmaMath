# SigmaMath - Geometry Quiz

A fun and interactive geometry quiz application.

## How to Deploy to GitHub Pages

1. **Create a Repository on GitHub**
   - Log in to your [GitHub account](https://github.com).
   - Click the **+** icon in the top right and select **New repository**.
   - Name it `SigmaMath` (or anything you like).
   - Keep it **Public**.
   - **Do not** check "Initialize this repository with a README" (we already have code).
   - Click **Create repository**.

2. **Push Your Code**
   - Copy the URL of your new repository (e.g., `https://github.com/YOUR_USERNAME/SigmaMath.git`).
   - Run the following commands in your terminal (replace the URL with yours):
     ```bash
     git remote add origin https://github.com/YOUR_USERNAME/SigmaMath.git
     git push -u origin main
     ```

3. **Enable GitHub Pages**
   - Go to your repository **Settings** tab.
   - Click on **Pages** in the left sidebar.
   - Under **Build and deployment** > **Source**, select **Deploy from a branch**.
   - Under **Branch**, select `main` and `/ (root)`.
   - Click **Save**.

4. **Visit Your Site**
   - Wait a minute or two.
   - Refresh the Pages settings page. You will see a link like `https://YOUR_USERNAME.github.io/SigmaMath/`.
   - Click it to see your live site!
